Cybernetic Support

This project describes a support model where inquiries are processed by an LLM but intermediate by a human as part of support for a trading system.

Requirements

  1. The component should provide an input for a client's request, which can be about an order, system behavior or documentation.
  2. It should provide a "Process Request" button which will send the request to the LLM.
  3. After processing, it should show the proposed action(s) it took, and the action results.
  4. The user can Accept, Modify, or Ignore the proposed action, which is logged as feedback to the designers of the LLM.
